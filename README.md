# Microsoft Website Clone

Projeto de recriação de interface inspirado no site oficial da Microsoft, desenvolvido com **HTML5**, **CSS3** e **BootStrap**, com foco em fidelidade visual, organização de layout e boas práticas de front-end.

## Objetivo

Este projeto foi desenvolvido com o objetivo de praticar:

* Reprodução de interfaces reais (UI Cloning)
* Estruturação semântica com HTML5
* Estilização moderna com CSS3
* Organização de layout e hierarquia visual
* Precisão na construção de interfaces baseadas em sites profissionais
* Utilização do framework BootStrap

## Tecnologias Utilizadas

* HTML5
* CSS3
* BootStrap

## Funcionalidades

* Navbar inspirada no padrão Microsoft
* Seções de destaque de produtos e serviços
* Layout organizado e bem distribuído
* Navegação simples e intuitiva
* Estrutura visual próxima ao site original

## Destaques Técnicos

* Uso de tags semânticas (`<header>`, `<main>`, `<section>`, `<footer>`, `<div>`)
* Estrutura baseada em **Flexbox** e/ou **CSS Grid**
* Separação clara entre HTML (estrutura) e CSS (estilo)
* Organização limpa e escalável do código
* Atenção a alinhamento, espaçamento e tipografia
